Binary Search Tree (BST)

What is a Binary Search Tree?

A Binary Search Tree (BST) is a hierarchical data structure where each node has at most two children, referred to as the left and right child.

The key property of BST is that the left subtree contains values less than the parent node, and the right subtree contains values greater than the parent node. This makes search, insertion, and deletion operations more efficient compared to linear data structures.

Inserting in BST

Insertion in a BST also follows the same principle as searching: - Start at the root. - Compare the new value. - Insert it into the left or right subtree recursively at the correct position. This maintains the BST property after each insertion.
